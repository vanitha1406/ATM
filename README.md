# Java ATM Simulation

## Overview

This is a console-based ATM Simulation project developed using Java. The application allows users to perform basic banking operations through a menu-driven interface.

## Features

* PIN Authentication
* Balance Inquiry
* Deposit Money
* Withdraw Money
* Transaction History
* Insufficient Balance Check
* Multiple Transactions Until Exit
* Account Lock After Invalid PIN Attempts

## Technologies Used

* Java
* Scanner Class
* ArrayList
* Loops and Conditional Statements

## How to Run

1. Compile the program:

javac ATM.java

2. Run the program:

java ATM

## Sample Operations

* Check Account Balance
* Deposit Amount
* Withdraw Amount
* View Transaction History
* Exit Application
